## Overview

This page serves as a centralized log for tracking client requests, feature asks, and documentation needs. It provides visibility into what clients are asking for and how those requests are being addressed.

---

## How to Submit a Request

<Steps>
  <Step title="Identify the Request">
    Clearly describe what you need — a new feature, documentation update, workflow change, or bug fix.
  </Step>
  <Step title="Submit via Preferred Channel">
    - **Email**: Send to your Village Labs contact
    - **Slack**: Post in your dedicated client channel
    - **In-Platform**: Use the feedback button in Peninsula
  </Step>
  <Step title="Include Context">
    Provide relevant details: use case, urgency, affected workflows, and any examples.
  </Step>
  <Step title="Track Status">
    Your request will appear in the log below with status updates as it progresses.
  </Step>
</Steps>

---

## Request Format

Each request is logged with the following information:

| Field | Description |
| --- | --- |
| **Date** | When the request was submitted |
| **Requestor** | Who submitted the request |
| **Description** | What is being requested |
| **Priority** | Critical / High / Medium / Low |
| **Status** | New / In Review / In Progress / Completed / Deferred |
| **Resolution** | How the request was addressed (when complete) |

## Review Cadence

Client requests are reviewed on a regular cadence:

| Review Type | Frequency | Participants |
| --- | --- | --- |
| **Triage** | Weekly | Village Labs implementation team |
| **Priority Review** | Bi-weekly | Client stakeholders + Village Labs |
| **Backlog Grooming** | Monthly | Full project team |

---

## Priority Definitions

| Priority | Definition | Target Response |
| --- | --- | --- |
| **Critical** | Blocking production work or causing data issues | Same day |
| **High** | Significant impact on daily workflows | Within 1 week |
| **Medium** | Improvement that would help but has workarounds | Within 1 month |
| **Low** | Nice-to-have enhancement | Backlog for future release |
