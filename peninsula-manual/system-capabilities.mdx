## Document Generation

Peninsula accelerates the creation of legal work product with AI-assisted drafting grounded in your firm's knowledge.

**Supported outputs:**

- Legal memos and opinion letters
- Due diligence summaries and issue spotting reports
- Transaction analyses and comparison charts
- Client correspondence and advisory letters
- Regulatory compliance checklists

**How it works:**

1. User initiates a drafting request with context and requirements
2. Kelso retrieves relevant precedents, authorities, and firm templates
3. AI generates a first draft with inline citations
4. Human reviewer edits, approves, or requests revisions
5. Final document is produced with full audit trail

<Note>
  All generated documents require human review before client delivery. Peninsula does not auto-publish legal work product.
</Note>

## File Editing

**Supported file types:**

| Format                   | Capabilities                                                     |
| ------------------------ | ---------------------------------------------------------------- |
| Spreadsheets (XLSX, CSV) | Update cells, add formulas, restructure data, generate summaries |
| Documents (DOCX)         | Edit text, apply formatting, insert sections, update tables      |
| PDFs                     | Extract content, annotate, convert to editable formats           |

**Example workflows:**

- Update a repurchase obligation spreadsheet with new participant data
- Revise a plan document section based on regulatory changes
- Extract key terms from a PDF agreement into a structured summary

## Search & Retrieval

Peninsula's search goes beyond keyword matching to deliver contextually relevant results from your entire knowledge corpus.

**Search capabilities:**

| Capability             | Description                                                         |
| ---------------------- | ------------------------------------------------------------------- |
| Hybrid retrieval       | Combines lexical (exact match) and semantic (concept match) search  |
| Metadata filtering     | Filter by matter, client, date, document type, confidentiality tier |
| Cross-system search    | Query across all connected source systems in one interface          |
| Relevance ranking      | Results ordered by authority, recency, and query fit                |
| Permission enforcement | Results filtered by user role and matter access before display      |

**ESOP-specific search examples:**

- Find all prior opinions on affiliated service group determinations
- Retrieve IRC 4975 prohibited transaction analyses from the last 3 years
- Search for ERISA fiduciary duty precedents across all matters
